# TravelBoost — Sistema de Diagnóstico para Agências de Viagens

Sistema completo de transformação digital para agências de viagens brasileiras.
Do diagnóstico ao conteúdo pronto em 1 hora.

## O que é

Ferramenta web (HTML/CSS/JS puro) que entrega:

- **Fase 1** — Diagnóstico com score 0–100 por dimensão (Presença, Conteúdo, Vendas, Tráfego, Conversão)
- **Fase 2** — Entregáveis prontos: vídeos recomendados, legendas, ofertas e checklist 30 dias
- **Fase 3** — Criação de artes com identidade visual do cliente *(em desenvolvimento)*
- **Fase 4** — Site do cliente com dashboard admin *(em desenvolvimento)*

## Como usar

Abra `sistema-agencia.html` diretamente no navegador. Sem dependências, sem servidor.

## Estrutura

```
sistema-agencia.html        ← App principal (Fases 1 e 2)
arquivos/                   ← Materiais de apoio e documentação
docs/                       ← Documentação técnica organizada
versions/                   ← Snapshots de versões anteriores
.kiro/skills/               ← Skills do agente Kiro
```
