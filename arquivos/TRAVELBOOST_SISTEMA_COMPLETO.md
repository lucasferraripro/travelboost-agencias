## 🎯 O SISTEMA TRAVELBOOST - Estrutura Completa

### FASE 1: DIAGNÓSTICO INTELIGENTE (15 min)
**O que faz:**
- Coleta dados da agência via formulário estruturado (via IA)
- Analisa perfil: tamanho, nichos, estágio, concorrentes, presença online
- Identifica 5-7 pontos críticos de melhoria
- Gera relatório visual com scorecard (0-100 por métrica)

**Dados coletados:**
- Nome/Localização/Faturamento atual
- Nichos principais (Praia, Aventura, Luxo, Trabalho, etc)
- Canais atuais (Instagram, WhatsApp, Google, Email)
- Problemas apontados (baixa conversão, falta de conteúdo, concorrência, etc)
- Base de clientes / ticket médio / ROI atual
- Equipe (solo vs times)

**Saída:**
```
SCORECARD DE DIAGNÓSTICO
📈 Presença Online: 35/100
💰 Monetização: 42/100
📱 Mobile Experience: 25/100
🎨 Identidade Visual: 15/100
📣 Marketing Ativo: 20/100
⚙️ Automação: 5/100
🎥 Conteúdo: 10/100

PONTOS DE MELHORIA (Prioridade):
1. Falta identidade visual marcante
2. Sem automação de leads (WhatsApp/Email)
3. Conteúdo escasso e desorganizado
4. Sem campanhas pagas estruturadas
5. Mobile experience ruim
6. Sem recomendações personalizadas
```

---

### FASE 2: IDENTIDADE VISUAL MARCANTE (20 min)
**O que faz:**
- Gera identidade visual completa via IA (cores, tipografia, estilo)
- Cria brand guidelines resumido
- Gera 5 variações de logo/marca
- Cria paleta visual com código hex específico
- Gera mockups de como a marca fica em Instagram, site, Stories

**Processo:**
1. IA analisa: nicho (praia/aventura/luxo), público-alvo, competitors
2. Gera **5 direções de marca completamente diferentes**
3. Agência escolhe 1 (ou pedindo fusão)
4. Sistema cria: logo, cores, tipografia, ícones, padrões, fotografia style

**Saída:**
```
BRAND IDENTITY KIT
- Logo Principal + Variações
- Paleta de Cores (Hex: #FF6B35, #004E89, #F7B801, #FFFFFF)
- Tipografia Principal: Montserrat (Headers) + Inter (Body)
- Estilo Fotográfico: Vibrante, people-centered, cinematic
- Ícones customizados (20+)
- Padrões geométricos
- Mockups: Instagram, Stories, Website Hero, Anúncio Facebook

+ Arquivo Figma editável
+ Guia de marca (PDF)
```

### FASE 4: PRODUÇÃO DE CONTEÚDO AUTOMÁTICA (20 min)
**O que faz:**
- Gera 30 posts (Instagram, TikTok, Reels) para próximos 30 dias
- Recomenda vídeos do seu acervo
- Cria copy específico para cada formato
- Agenda tudo em Canva / Buffer / Later

**Tipos de Conteúdo Gerado:**
1. **Posts Educativos** (5/semana) - "Dicas de viagem", "Como economizar", guias
2. **Posts de Testemunho** (2/semana) - Histórias de clientes (hero's journey)
3. **Posts Promocionais** (2/semana) - Ofertas, pacotes, promoções sazonais
4. **Posts de Engagement** (3/semana) - Enquetes, perguntas, storytelling
5. **Reels Curtos** (4/semana) - 15-30s sobre destinos, dicas rápidas
6. **Stories Diários** (6/semana) - Behind-the-scenes, promoções urgentes

**Exemplo de Post Gerado:**

```
FORMATO: Instagram Reel (15s)
TEMA: "Destinos que custam menos que você pensa"

COPY:
"Heading: ✈️ Viagem de sonho, preço real
Body: Vietnã, Tailândia, México... países incríveis e seus preços? Surpreendentes. 
Clica pra descobrir quanto custa uma semana em cada lugar 👇

CTA: "Manda um DM! Tenho pacotes que cabem no seu bolso."

VISUAL: 
- Template Canva (marca visual definida)
- 3 vídeos curtos do acervo (15s cada)
- Transições suaves entre destinos
```

### FASE 5: CAMPANHAS ONLINE ESTRUTURADAS (20 min)
**O que faz:**
- Cria 3-5 campanhas pré-prontas (Meta Ads, Google Ads, TikTok)
- Estrutura públicos-alvo (lookalike, interesse, custom audience)
- Gera copy + criativos para cada anúncio
- Define budget recomendado
- Cria sequência de follow-up (Email + WhatsApp)

**Campanhas Pré-Estruturadas:**

#### Campanha 1: "Viagem dos Sonhos" (Awareness)
```
Público: Mulheres 25-45, Classe A/B, Interesse: Viagem, Lifestyle
Objetivo: Alcance (brand awareness)
Budget Recomendado: R$ 300-500/mês
Duração: Contínua

Criativos:
- Vídeo 1: "Você tem medo de viajar porque acha caro?"
- Vídeo 2: "Destinos TOP que ninguém sabe"
- Vídeo 3: Testemunho de cliente (história de transformação)

Copy:
"Quer viajar mas acha que vai falir? Fale com a gente. Já ajudamos 200+ pessoas a 
encontrar viagens incríveis (e baratas). Spoiler: a sua próxima viagem sai mais barato 
que você imagina. 🌍✈️"

CTA: "Agende uma conversa gratuita"
```

#### Campanha 2: "Garantia de Resultado" (Consideration)
```
Público: Warm audience (quem visitou site/Instagram/clicou anúncio)
Objetivo: Conversão
Budget: R$ 200-300/mês
Duração: 14 dias (retargeting)

Copy:
"Já viu a gente em algum lugar? Ótimo. Porque queremos te mostrar por que a gente 
é diferente. Não vendemos viagens. A gente cria histórias. Vem conversar 👇"

CTA: "Quero entender melhor"
```

#### Campanha 3: "Oferta Relâmpago" (Conversion)
```
Público: Hot audience (já interagiu 3+ vezes)
Objetivo: Venda direta
Budget: R$ 150-250/mês
Duração: 7 dias (urgência)

Copy:
"ÚLTIMA CHANCE: Só até amanhã você leva 15% de desconto em viagens para Maldivas. 
200 pessoas já aproveitaram. Você será a 201ª? 🏝️⏰"

CTA: "Aproveitar oferta agora"
```
