## As 5 Campanhas Pré-Estruturadas

### Campanha 1 — "Viagem dos Sonhos" (Awareness)
```
Objetivo: Alcance / Brand Awareness
Público: Mulheres 25–50, Classe A/B, Interesse: Viagem, Lifestyle
Budget: R$ 300–500/mês
Duração: Contínua (sempre ligada)

HEADLINE (30 chars):
"Você tem medo de viajar porque acha caro?"

BODY (300 chars):
"A gente ajudou 200+ pessoas a encontrar viagens incríveis (e baratas). 
Spoiler: a sua próxima viagem sai mais barato que você imagina. 🌍✈️"

CTA BOTÃO: "Agendar conversa gratuita"

CRIATIVOS NECESSÁRIOS:
- Vídeo 1: "Você tem medo de viajar porque acha caro?" (30s)
- Vídeo 2: "Destinos TOP que ninguém sabe" (30s)
- Vídeo 3: Testemunho de cliente (30s)

PÚBLICO ESTRATIFICADO:
- Frio: Interesse geral em viagem + lifestyle
- Morno: Quem visitou site/Instagram recentemente
- Quente: Quem interagiu 3+ vezes
```

### Campanha 2 — "Garantia de Resultado" (Consideration)
```
Objetivo: Conversão (levar para consulta)
Público: Warm audience (visitou site/Instagram)
Budget: R$ 200–300/mês
Duração: 14 dias (retargeting)

HEADLINE:
"Já viu a gente em algum lugar?"

BODY:
"Ótimo. Porque queremos te mostrar por que somos diferentes. 
Não vendemos viagens. A gente cria histórias. Vem conversar 👇"

CTA BOTÃO: "Quero entender melhor"
```

### Campanha 3 — "Oferta Relâmpago" (Conversion)
```
Objetivo: Venda direta
Público: Hot audience (interagiu 3+ vezes)
Budget: R$ 150–250/mês
Duração: 7 dias (urgência)

HEADLINE:
"ÚLTIMA CHANCE: 15% OFF em [DESTINO]"

BODY:
"Só até amanhã você leva 15% de desconto em viagens para [DESTINO]. 
200 pessoas já aproveitaram. Você será a 201ª? 🏝️⏰"

CTA BOTÃO: "Aproveitar oferta agora"
```

### Campanha 4 — "Prova Social" (Trust)
```
Objetivo: Construir confiança
Público: Frio + Morno
Budget: R$ 100–200/mês
Duração: Contínua

HEADLINE:
"[Nome] foi para [Destino] e voltou assim:"

BODY:
"[Depoimento real do cliente]. Quer sua história de viagem também? 
Clica no link da bio 👇"

CTA BOTÃO: "Ver mais histórias"

CRIATIVO: Vídeo de depoimento de cliente (30–60s)
```

### Campanha 5 — "Destino do Mês" (Seasonal)
```
Objetivo: Venda de pacote específico
Público: Interesse no destino específico
Budget: R$ 200–400/mês
Duração: 30 dias (rotativo por mês)

HEADLINE:
"[DESTINO] em [MÊS]: tudo que você precisa saber"

BODY:
"Clima, preço, o que fazer e onde ficar. Montamos o roteiro perfeito 
para você. Pacotes a partir de R$ [PREÇO]. 📍"

CTA BOTÃO: "Ver pacote completo"
```

## Tutorial: Subir Campanha no Meta Ads

### Passo a Passo

**1. Acessar o Gerenciador de Anúncios**
```
→ facebook.com/adsmanager
→ Criar nova campanha
```

**2. Escolher Objetivo**
```
Awareness → Alcance de marca
Consideration → Tráfego / Engajamento
Conversion → Conversões / Mensagens
```

**3. Configurar Conjunto de Anúncios**
```
→ Definir público (usar dados acima)
→ Definir orçamento diário (R$ 10–50/dia)
→ Definir período (7, 14 ou 30 dias)
→ Definir posicionamentos (automático recomendado)
```

**4. Criar Anúncio**
```
→ Subir criativo (vídeo ou imagem)
→ Colar headline (copy pronto acima)
→ Colar body text (copy pronto acima)
→ Definir CTA (botão)
→ Definir URL de destino (site ou WhatsApp)
```

**5. Revisar e Publicar**
```
→ Verificar preview (mobile + desktop)
→ Confirmar orçamento
→ Publicar
→ Aguardar aprovação (1–24h)
```

## Checklist de Lançamento de Campanha

### Antes de Subir
- [ ] Criativo pronto (vídeo ou imagem 1080×1080 ou 1080×1920)
- [ ] Copy escrito (headline + body + CTA)
- [ ] Público definido (interesses + comportamentos)
- [ ] Orçamento definido (mínimo R$ 10/dia)
- [ ] URL de destino funcionando (site ou WhatsApp)
- [ ] Pixel do Facebook instalado no site
- [ ] Conta de anúncios verificada

### Após Publicar
- [ ] Aguardar aprovação (1–24h)
- [ ] Verificar se está rodando (status "Ativo")
- [ ] Monitorar primeiras 48h (CPM, CTR, CPC)
- [ ] Ajustar se necessário (pausar criativos ruins)

### Métricas para Monitorar
| Métrica | Meta | Ação se Abaixo |
|---------|------|----------------|
| CTR | > 1% | Trocar criativo |
| CPC | < R$ 2,00 | Ajustar público |
| CPM | < R$ 20,00 | Ampliar público |
| ROAS | > 3x | Escalar budget |

## Criação de Oferta para Anúncio

### Fórmula de Oferta que Converte
```
[DESTINO] + [PERÍODO] + [PREÇO] + [PARCELAMENTO] + [URGÊNCIA] + [CTA]

Exemplo:
"Natal - RN | 5 noites | A partir de R$ 1.800/pessoa
Parcelamos em 10x sem juros | Vagas limitadas!
📲 Fale agora: [WhatsApp]"
```

### Elementos Obrigatórios
1. **Destino claro** — onde vai
2. **Duração** — quantas noites
3. **Preço** — a partir de quanto
4. **O que inclui** — aéreo, hotel, café...
5. **Parcelamento** — reduz objeção de preço
6. **Urgência** — vagas limitadas, prazo
7. **CTA** — o que fazer agora

### Teste A/B de Ofertas
Sempre testar 2 versões:
- **Versão A**: Foco no preço ("A partir de R$ 1.800")
- **Versão B**: Foco no destino ("Natal te espera")

Rodar por 7 dias, pausar a que performa pior, escalar a melhor.

## Ciclo de Vendas Contínuo

Para ter sempre vendas e vendas e vendas:

```
SEMANA 1: Lançar campanha de awareness
SEMANA 2: Ativar retargeting para quem viu
SEMANA 3: Oferta relâmpago para quem interagiu
SEMANA 4: Analisar, otimizar, escalar o que funcionou
SEMANA 5: Novo destino, novo criativo, novo ciclo
```

### Rotação de Criativos
- Trocar criativos a cada 2–3 semanas (evitar fadiga)
- Sempre ter 3–5 criativos rodando em paralelo
- Pausar criativos com CTR < 0.5%
- Escalar criativos com CTR > 2%
