# 🚀 Prompts Mestre — NanoBanana AI

Estes são os prompts oficiais integrados ao sistema TravelBoost para geração de artes de alta conversão.

## Variáveis do Sistema
O sistema substitui automaticamente as seguintes tags antes de enviar para a IA:
- `[COR_PRIMARIA]`, `[COR_SECUNDARIA]`, `[COR_TEXTO_DESTAQUE]`
- `[NOME_PROMOCAO]`, `[SUBTITULO_1]`, `[SUBTITULO_2]`
- `[LISTA_DESTINOS]`, `[DESTINO_PRINCIPAL]`
- `[VALOR]`, `[CHAMADA_PRECO]`, `[CHAMADA_ACAO]`

---
