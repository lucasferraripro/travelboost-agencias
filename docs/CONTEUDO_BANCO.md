```

[CORPO DO TEXTO — 2-3 linhas]

[BENEFÍCIO OU PROVA SOCIAL]

[CTA CLARO]

[HASHTAGS — 3-5]
```

### Tipos de Legenda

#### Tipo 1 — Oferta Direta (40 legendas)
Foco: preço + destino + parcelamento + urgência

#### Tipo 2 — Educativa (40 legendas)
Foco: dica de viagem + informação útil + CTA suave

Foco: história de cliente + transformação + prova social
